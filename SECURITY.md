# 🔐 Security Practices

Cosmic Life Companion follows these security practices:

- Environment variables for API keys
- Firebase authentication handling
- Secure deployment through Vercel
- Dependency monitoring
- No sensitive credentials stored in source code

---

# ⚠️ Responsible Disclosure

Please do not publicly disclose vulnerabilities before they are reviewed and addressed.

We appreciate responsible security research that helps improve the project.
